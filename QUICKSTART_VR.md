# PSE in VR - Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Open the VR Page
Visit: [chemie-lernen.org/pages/pse-in-vr/](/pages/pse-in-vr/)

### Step 2: Click "VR starten"
Click the purple button to launch the VR experience

### Step 3: Start Exploring!
- **With VR Headset:** Put on your headset and click "Enter VR"
- **Without VR:** Click in the window, then use WASD + Mouse to explore

## ✨ Features to Try

1. **Explore Elements:** Click/touch any element to see details
2. **Change Rooms:** Press N to visit different learning spaces
3. **Rotate View:** Use mouse/controller to look around
4. **Read Info:** Element cards show atomic number, mass, and more
5. **Group Elements:** Notice color coding by element groups

---

## 💡 Tips

### First Time Users
- Start without VR headset to get familiar
- Take 5 minutes to explore the landing room
- Read the help overlay (bottom-left corner)
- Try all control keys before putting on headset

### Best Experience
- Use Chrome or Edge browser
- Ensure good internet connection
- Close other browser tabs
- Enable hardware acceleration
- Use headphones for spatial audio (if available)

### Troubleshooting
- **VR button missing?** Update browser or enable WebXR
- **Can't move?** Click in the window to activate controls
- **Laggy?** Close other tabs, check internet speed
- **Black screen?** Wait for loading (up to 10 seconds)

## 🎓 Learning Path

### Beginner (15 min)

2. Explore the landing room
3. Click 5 different elements
4. Practice desktop controls

1. Complete beginner path
2. Visit 3 different rooms (press N)
3. Find all alkali metals
4. Try VR mode (if headset available)

### Advanced (60 min)
1. Complete intermediate path
2. Explore all available rooms
3. Identify element groups by color
4. Use VR controllers for interaction

## ❓ FAQ

### Q: Do I need a VR headset?
**A:** No! You can use your mouse and keyboard on any computer.

### Q: Is it free?
**A:** Yes, completely free. No installation required.

### Q: Can I use this on a phone?
**A:** Yes, but a computer with mouse/keyboard provides the best experience.

### Q: Which VR headsets work?
**A:** Meta Quest, HTC Vive, Valve Index, and most WebXR-compatible headsets.

### Q: Can I use this offline?
**A:** After first load, the service worker enables limited offline functionality.

### Q: Is my data collected?
**A:** Only anonymous analytics for improvement. No personal data is stored.
