Hess-Gesetz verstehen

📐 Grundprinzip

Das Hess-Gesetz (1840) besagt, dass die Enthalpieänderung einer chemischen Reaktion unabhängig vom Reaktionsweg ist.

ΔH = Σ ΔHReaktionen

  • Die Enthalpie ist eine Zustandsgröße
  • Nur der Anfangs- und Endzustand sind relevant
  • Der gewählte Reaktionsweg spielt keine Rolle

🔄 Anwendungen

Hess-Gesetz wird verwendet, wenn:

  • Direkte Messung unmöglich: Reaktion ist zu schnell oder gefährlich
  • Zwischenprodukte unbekannt: Keine experimentellen Daten verfügbar
  • Theoretische Werte: Berechnung aus Bildungsenthalpien

Beispiel: Kohlenstoffverbrennung kann über verschiedene Zwischenstufen berechnet werden.

⚗️ Born-Haber-Kreisprozess

Ein spezielles Anwendungsbeispiel des Hess-Gesetzes:

  • Berechnung der Gitterenergie von Salzen
  • Kombination von thermochemischen Daten
  • Zyklischer Prozess aus mehreren Schritten

ΔHLatt = ΣΔHSchritte

🔬 Methanverbrennung

Beispiel: CH₄ + 2O₂ → CO₂ + 2H₂O

Weg A (direkt):

CH₄ + 2O₂ → CO₂ + 2H₂O
ΔH = -890.3 kJ/mol

Weg B (über Zwischenprodukte):

CH₄ → C + 2H₂, ΔH₁ = +74.8 kJ/mol
C + O₂ → CO₂, ΔH₂ = -393.5 kJ/mol
2H₂ + O₂ → 2H₂O, ΔH₃ = -571.6 kJ/mol
ΣΔH = 74.8 - 393.5 - 571.6 = -890.3 kJ/mol

Ergebnis: Beide Wege ergeben das gleiche Ergebnis (ΔH = -890.3 kJ/mol)

✅ Bestätigt das Hess-Gesetz
