⚖️ Das Gesetz der Erhaltung der Masse

In einer chemischen Reaktion wird keine Masse erzeugt oder vernichtet. Die Anzahl der Atome jedes Elements muss auf beiden Seiten der Gleichung gleich sein.

Beispiel:

2H₂ + O₂ → 2H₂O

4 H-Atome und 2 O-Atome auf jeder Seite

🔢 So wird ausgeglichen

  1. Zählen Sie die Atome jedes Elements auf beiden Seiten
  2. Ändern Sie Koeffizienten (Zahlen vor Formeln), nicht Indizes
  3. Beginnen Sie mit dem komplexesten Molekül
  4. Balancieren Sie zuletzt Elemente, die alleine vorkommen (O₂, H₂, etc.)
  5. Überprüfen Sie, ob alle Atomzahlen übereinstimmen

💡 Tipps & Tricks

  • Behandeln_polyatomare Ionen_als_Einheit (z.B. SO₄²⁻, NO₃⁻)
  • Wenn_polyatomare Gruppen_auftauchen, sparen Sie Zeit
  • Überprüfen Sie Ihre Arbeit durch Abzählen
  • Reine Elemente (O₂, H₂, N₂) sollten Sie zuletzt balancieren
  • Kleine ganze Zahlen bevorzugen (durch 2, 3 etc. teilen wenn möglich)

🎯 Warum ist Ausgleichen wichtig?

  • Stöchiometrische Berechnungen: Erforderlich für Mengenberechnungen bei chemischen Reaktionen
  • Reaktionsausbeute: Ermöglicht Vorhersage der Produktmengen
  • Laborpraxis: Bestimmt, wie viel Edukt für eine gewünschte Produktmenge benötigt wird
  • Industrielle Prozesse: Optimiert Rohstoffnutzung und Kosten
  • Umweltanalyse: Berechnet Abfallprodukte und Emissionen
