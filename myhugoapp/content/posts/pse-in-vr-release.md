Wir freuen uns, die Veröffentlichung von **dem PSE in VR** ankündigen zu können - eine virtuelle Realität-Erfahrung für das Periodensystem der Elemente. Diese Innovation repräsentiert einen bedeutenden Schritt in der digitalen Chemiebildung.

## 🌟 Was ist neu?

Das PSE in VR ist eine browserbasierte Anwendung, die das Periodensystem der Elemente in einem immersiven 3D-Raum präsentiert.

### Kernfunktionen

- **🥽 Immersive 3D-Umgebung:** Erkunden Sie alle 118 Elemente in einem virtuellen Raum
- **🎮 Dual-Modus-Steuerung:**
  - VR-Modus für Headsets wie Meta Quest, HTC Vive, Valve Index
  - Desktop-Modus mit Maus und Tastatur
- **👥 Multiplayer-Unterstützung:** Lernen Sie gemeinsam mit anderen in virtuellen Räumen
- **⚗️ Gefahrfreie Experimente:** Simulieren Sie gefährliche chemische Reaktionen sicher
- **📱 Browser-basiert:** Keine Installation erforderlich - funktioniert direkt im Webbrowser

## 🎯 Pädagogischer Nutzen

### Für Schüler

- **Visuelles und räumliches Lernen:** 3D-Darstellungen fördern das räumliche Verständnis chemischer Konzepte
- **Aktive Exploration:** Statt passiv zuzuhören, erkunden Schüler den Lernstoff aktiv
- **Steigerung der Motivation:** Gamification-Elemente machen Chemie-Lernen spannend
- **Selbstgesteuertes Lernen:** Schüler können in ihrem eigenen Tempo erkunden

### Für Lehrer

- **Innovative Unterrichtsmethoden:** Integration moderner Technologie in den Chemieunterricht
- **Sichere Experimente:** Gefährliche Reaktionen können ohne Risiko simuliert werden
- **Kollaboratives Lernen:** Mehrere Schüler können gemeinsam im VR-Raum arbeiten
- **Flexible Integration:** Funktioniert im Klassenzimmer mit Desktops oder mit VR-Headsets

## 🚀 Loslegen

### Schnellstart

1. Besuchen Sie [pse.chemie-lernen.org](https://pse.chemie-lernen.org/)
2. Klicken Sie auf "VR starten"
3. Bei VR-Headset: Klicken Sie auf "Enter VR"
4. Ohne VR-Headset: Nutzen Sie die Maussteuerung

### Steuerung

**VR-Controller:** Trigger zum Auswählen, Joystick zur Fortbewegung

**Desktop:** Maus zum Umsehen, Tasten zur Steuerung

## 🎮 Feature-Highlights

- **Adaptive UI:** Das Interface passt sich automatisch an VR- oder Desktop-Modus an
- **Interaktive Elemente:** Klicken Sie auf Elemente für detaillierte Informationen
- **Multiplayer-Räume:** Lernen Sie gemeinsam mit Chat-Funktion
- **Virtuelle Labore:** Experimente mit Alkalimetallen, reaktiven Nichtmetallen und radioaktiven Prozessen

Dieses Projekt ist Teil einer interdisziplinären Forschungskooperation zur Untersuchung der Wirksamkeit von VR in der Chemiebildung.

Unsere Pilotstudien zeigen:

- **Steigerung der Behaltensrate:** +23% im Vergleich zu traditionellen Methoden
- **Höhere Engagement:** 89% der Schüler berichten von höherer Motivation
- **Besseres räumliches Verständnis:** +34% bei Tests zu molekularen Strukturen
