# About learn-chemistry.org

learn-chemistry.org is a free, open educational platform for chemistry. We provide interactive tools, 3D visualizations, and immersive VR experiences to make chemistry accessible and engaging.

## Features

- 3D Periodic Table with 118 elements
- VR element rooms
- Interactive calculators (pH, molar mass, stoichiometry, and more)
- Quizzes and flashcards
- Learning paths for all levels
