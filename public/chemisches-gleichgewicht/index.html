⚖️ Dynamisches Gleichgewicht

Ein chemisches Gleichgewicht ist dynamisch, wenn die Hinreaktionsgeschwindigkeit gleich der Rückreaktionsgeschwindigkeit ist:

  • vhin = vrück
  • Die Konzentrationen aller Reaktionspartner bleiben konstant
  • Kc = [Produkte]/[Edukte]

📐 Le Chatelier'sches Prinzip

Wenn ein System im Gleichgewicht gestört wird, reagiert es, um das Gleichgewicht wiederherzustellen:

Einflüsse auf das Gleichgewicht:

  • Temperaturerhöhung: Favors endotherme Reaktion
  • Druckerhöhung: Favors Seite mit weniger Gasvolumen
  • Konzentrationserhöhung: Favors Produkte
  • Konzentrationserniedrigung: Favors Edukte

🔬 Gleichgewichtskonstante

Die Gleichgewichtskonstante Kc gibt die Position des Gleichgewichts an:

  • Kc >> 1: Gleichgewicht liegt auf Produktseite
  • Kc << 1: Gleichgewicht liegt auf Eduktseite
  • Kc = 1: Edukte und Produkte im gleichen Verhältnis
Beispiel: A + B → C + D

Kc = 1: Wenn [A] = [B] = [C] = [D] = 1.0 M, dann ist das System im Gleichgewicht.

Kc = 10: Bei Konzentration [C] = [D] = 10.0 M, verschiebt sich das Gleichgewicht stark zu Produkten.

📊 Reaktionsquotient ξ

Der Reaktionsfortschritt ξ gibt an, wie weit die Reaktion fortgeschritten ist:

ξ = ([Produkte] / [Edukte])1/n - 1

  • ξ = 0: Reaktion noch nicht gestartet
  • 0 < ξ < 1: Reaktion läuft, aber noch nicht im Gleichgewicht
  • ξ = 1: Reaktion abgeschlossen, maximale Ausbeute erreicht
Beispielrechnung:

Für A + B → C + D mit Kc = 1:

Schritt 1: ξ₀ = ([A]₀/[B]₀) - 1 = 1/1 - 1 = 0
Schritt 2: ξ₀ × (1 - ξ₀) = 0 × (1 - 0) = 0
Schritt 3: ξ = ξ₀ × exp(-kt) = 0 × exp(-0) = 0
Ergebnis: Die Reaktion befindet sich noch am Startpunkt.

🔬 Gibbs-Energie

Die Gibbs-Energie ΔG bestimmt, ob eine Reaktion spontan abläuft:

ΔG = ΔH - T × ΔS

  • ΔG < 0: Reaktion ist exotherm (spontan)
  • ΔG > 0: Reaktion ist endotherm (nicht spontan)
  • ΔG = 0: Reaktion ist im Gleichgewicht

Bei Gleichgewicht:

ΔG = 0

Da ΔH = -TΔS (enthalpische Änderung kompensiert Entropieänderung)

⚗️ Industrielle Anwendungen

Praktische Beispiele für Gleichgewichte in der Industrie:

  • Ammoniak-Synthese (Haber-Bosch): N₂ + 3H₂ → 2NH₃, Kc ~ 10⁴ bis 10⁵
  • Wassergas-Gleichgewicht: H₂ + I₂ ⇌ 2HI, Kc ~ 50 bis 70
  • Eisenoxid-Reduktion: Fe₂O₃ + 3CO → 2Fe + 3CO₂, Hochtemperatur-Anwendung
  • Photosynthese in Pflanzen: CO₂ + H₂O ⇌ C₆H₁₂O₆ + O₂, Lichtabhängig

🧪 Biochemische Gleichgewichte

Gleichgewichte in biologischen Systemen:

  • Blut-Puffer: H₂CO₃/HCO₃⁻-System, pH-Regulation
  • ATP-Synthese: Adenosintriphosphat + Pyrophosphat → ATP + PPi, Zelluläres Gleichgewicht
  • Hämoglobin-Dissoziation: Hb + O₂ ⇌ HbO₂ + O₂, Sauerstofftransport
  • Enzym-Katalyse: Michaelis-Menten-Gleichgewicht, Enzymsättigung

🌍️ Umweltchemie

Gleichgewichte in Atmosphäre und Ozeanen:

  • CO₂-Absorption: CO₂ + H₂O ⇌ H₂CO₃, Ozeanischer Kohlenstoffkreislauf
  • NOₓ/NO₂-Gleichgewicht: N₂ + O₂ ⇌ 2NO, Smog-Bildung, Autoabgase
  • O₃/O₂-Gleichgewicht: 2O₃ ⇌ 3O₂, Ozonzerstörung und Schutzschicht
  • Ozeanversauerung: 2H₂O + O₂ ⇌ 2H₂O + O₂, Photosynthese
